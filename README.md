# devopsjr-api-cicd

Projeto de portfólio DevOps Jr com foco em construção de uma API simples, containerização com Docker e automação inicial com GitHub Actions.

## Objetivo
Criar uma API mínima com FastAPI para demonstrar organização de projeto, execução local, empacotamento com Docker e base para CI/CD.

## Escopo inicial
- endpoint `/health`
- endpoint `/info`
- execução local com uvicorn
- containerização com Docker
- pipeline CI inicial

## Stack
- Python 3
- FastAPI
- Uvicorn
- Docker
- GitHub Actions

## Estrutura inicial
- `app/` código da aplicação
- `tests/` testes
- `.github/workflows/` automações CI
